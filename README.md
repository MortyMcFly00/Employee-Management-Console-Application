# Employee Management Console Application

Employee Management Console Application is a Python command-line project that allows users to create, view, search, edit, import, and export employee records. The application uses a menu-driven interface to manage employee information in a structured way.

## Project Overview

This project was created as a school assignment to practice building a multi-function console application in Python. The program manages employee records by allowing users to enter employee information, display records, search by SSN, edit existing records, and save or load data from a text file.

The project focuses on structured program flow, user input handling, and organizing shared application state across multiple functions.

## Tech Used

- Python

## Features

- Menu-driven command-line interface
- Add employee records with fields such as name, SSN, phone, email, and salary
- View all employee records in a formatted list
- Search for employees by SSN
- Edit existing employee records
- Import employee data from a text file
- Export employee data to a text file
- Maintains employee IDs across records

## File Structure

- `main.py` — runs the main menu and application flow
- `functions.py` — contains the core employee management functions
- `shared_objects.py` — stores shared variables and application state
